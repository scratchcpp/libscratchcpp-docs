Image formats

To work with costumes, libscratchcpp has to read image files from Scratch projects. Only JPEG and PNG are currently supported, but SVG is going to be supported in the future.

Implementing a custom image format

To implement a custom image format that libscratchcpp doesn't support, subclass IImageFormat and override all of the methods.

Then subclass IImageFormatFactory and override createInstance() .

It's recommended to use the libscratchcpp namespace like this in your classes:

using namespace libscratchcpp;

Here's an example of the createInstance() method:

std::shared_ptr<IImageFormat> TiffImageFormatFactory::createInstance() {
return std::make_shared<TiffImageFormat>();
}

Registering the image format

To register the image format, you need to register the factory class. It will be used by classes such as Costume internally to instantiate your image format.

libscratchcpp::ScratchConfiguration::registerImageFormat("tiff", std::make_shared<TiffImageFormatFactory>());
