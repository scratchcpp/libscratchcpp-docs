◆ stop()

void VirtualMachine::stop ( bool savePos = true,
bool breakFrame = false,
bool goBack = false )

Use this to stop the script from a function.

Parameters
[in]savePosChanges the return value of savePos().
[in]breakAtomicWhether to break the frame after stopping the script.
[in]goBackWhether to go back so that the current instruction can run again in the future.
Note
If the script is set to run without screen refresh, the VM won't stop. The only parameter which won't be ignored is goBack.
